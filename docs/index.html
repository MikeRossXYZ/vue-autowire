# Introduction

In development!

Vue Autowire is a Vue plugin with conventions for automatically wiring Vue assets:

  • components
  • views
  • directives
  • filters
  • mixins
  • routes

First time here? Read how to get it installed on your project, then read through the conventions used to register the different assets.
